Register Hangouts extension

This Chrome extension uses registerProtocolHandler() to direct tel: links for phone numbers to Google Hangouts. First go to chrome://settings/handlers (Chrome security requires that you copy and paste the link in a new tab) and remove any existing tel: entry using its 3-dot menu. Then click the register button below and allow RegisterHangouts to open tel links in the security popup.

The extension icon incorporates a phone silhouette created by Dave Gandy under the Creative Commons Attribution-Share Alike 3.0 Unported license.
